## Key Architectural Decisions

1. **Unified Rendering** (ADR-0028)
   - SOV and PPTX visualization contracts now extend shared RenderSpec
   - Enables SOV â†’ PPTX chart flow without translation

2. **Cross-Platform Concurrency** (ADR-0012)
   - Spawn-safe process pools for Windows/macOS/Linux
   - Shell command abstraction with OS detection
   - No raw multiprocessing/threading allowed

3. **Contract Discipline** (ADR-0009)
   - Pydantic models as Tier-0 source of truth
   - JSON Schema auto-generation
   - CI enforcement of contract compliance
