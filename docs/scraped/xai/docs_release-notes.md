Stay up to date with the latest changes to the xAI API.

# [November 2025](https://docs.x.ai/docs/release-notes#november-2025)

### [Grok 4.1 Fast is available in Enterprise API](https://docs.x.ai/docs/release-notes#grok-41-fast-is-available-in-enterprise-api)

You can now use Grok 4.1 Fast in the [xAI Enterprise API](https://x.ai/api). For more details, check out [our blogpost](https://x.ai/news/grok-4-1-fast).

### [Agent tools adapt to Grok 4.1 Fast models and tool prices dropped](https://docs.x.ai/docs/release-notes#agent-tools-adapt-to-grok-41-fast-models-and-tool-prices-dropped)

- You can now use Grok 4.1 Fast models with the agent tools, check out the [documentation of agent tools](https://docs.x.ai/docs/guides/tools/overview) to get started.
- The price of agent tools drops by up to 50% to no more than $5 per 1000 successful calls, see the new prices at [the pricing page](https://docs.x.ai/docs/models#tools-pricing).

### [Files API is generally available](https://docs.x.ai/docs/release-notes#files-api-is-generally-available)

You can now upload files and use them in chat conversations with the Files API. For more details, check out [our guide on Files](https://docs.x.ai/docs/guides/files).

### [New Tools Available](https://docs.x.ai/docs/release-notes#new-tools-available)

- **Collections Search Tool**: You can now search through uploaded knowledge bases (collections) in chat conversations via the API. For more details, check out the [docs](https://docs.x.ai/docs/guides/tools/collections-search-tool).
- **Remote MCP Tools**: You can now use tools from remote MCP servers in chat conversations via the API. For more details, check out the [docs](https://docs.x.ai/docs/guides/tools/remote-mcp-tools).
- **Mixing client-side and server-side tools**: You can now mix client-side and server-side tools in the same chat conversation. For more details, check out the [docs](https://docs.x.ai/docs/guides/tools/advanced-usage#mixing-server-side-and-client-side-tools).

# [October 2025](https://docs.x.ai/docs/release-notes#october-2025)

### [Tools are now generally available](https://docs.x.ai/docs/release-notes#tools-are-now-generally-available)

New agentic server-side tools including `web_search`, `x_search` and `code_execution` are available. For more details, check out [our guide on using Tools](https://docs.x.ai/docs/guides/tools/overview).

# [September 2025](https://docs.x.ai/docs/release-notes#september-2025)

### [Responses API is generally available](https://docs.x.ai/docs/release-notes#responses-api-is-generally-available)

You can now use our stateful Responses API to process requests.

### [Structured Outputs We're excited to announce that Grok now supports structured outputs. Grok can](https://docs.x.ai/docs/release-notes#structured-outputs-were-excited-to-announce-that-grok-now-supports-structured-outputs-grok-can)

now format responses in a predefined, organized format rather than free-form text. 1. Specify the desired schema

Text

```
{
    "name": "movie_response",
    "schema": {
        "type": "object",
        "properties": {
            "title": { "type": "string" },
            "rating": { "type": "number" },
        },
        "required": [ "title", "rating" ],
        "additionalProperties": false
    },
    "strict": true
}
```

1. Get the desired data

Text

```
{
  "title": "Star Wars",
  "rating": 8.6
}
```

Start building more reliable applications. Check out the [docs](https://docs.x.ai/docs/guides/structured-outputs#structured-outputs) for more information.
